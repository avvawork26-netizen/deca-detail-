# Deca Detail — Car Detailing Booking Site

Professional car detailing booking website for Deca Detail (Orlando, FL).

## Services
- Exterior Wash — $25
- Interior Detail — $25
- Wax — $20
- Mobile service or drop-off available

## Live Site
Auto-deploys to Netlify on every push to main.

## Admin Panel
Visit the live URL with ?admin=YOUR_KEY to access the bookings dashboard.

## Deploy Workflow
After making changes, run: git add . && git commit -m "message" && git push
Netlify will auto-deploy within ~30 seconds.

## Setup
- Hosting: Netlify (free tier)
- Version control: GitHub
- Stack: Single-file HTML/CSS/JS
